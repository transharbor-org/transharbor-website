The official website for TransHarbor, a safe and welcoming community chat platform for transgender individuals and allies.

## About TransHarbor

TransHarbor provides a secure, supportive online space where transgender people and allies can connect, share experiences, and access valuable resources. Our community is built on principles of respect, inclusivity, and mutual support.

## Website Features

- **Community Information**: Learn about our mission, values, and guidelines
- **Discord Integration**: Easy access to join our chat server
- **Resource Hub**: Comprehensive collection of helpful resources
- **Regular Updates**: Community news and announcements
- **Mobile-Friendly**: Responsive design for all devices

## Getting Started

### For Community Members
1. Visit [transharbor.org](https://transharbor.org)
2. Read our [Community Guidelines](/guidelines)
3. [Join our Discord server](/chat)
4. Explore our [Resources](/resources)

### For Contributors
This website is built with Jekyll and hosted on GitHub Pages.

#### Local Development
```bash
# Clone the repository
git clone https://github.com/transharbor/transharbor-website.git
cd transharbor-website

# Install dependencies
bundle install

# Serve locally
bundle exec jekyll serve

# Visit http://localhost:4000
```

#### Making Changes
1. Fork the repository
2. Create a feature branch
3. Make your changes
4. Test locally
5. Submit a pull request
